Este projeto é o meu primeiro site de portfólio, desenvolvido com HTML e CSS para apresentar quem sou, minhas habilidades, experiências e trajetória na área de tecnologia. O site é composto por três páginas principais:

Home: Introdução, foto, habilidades e links diretos para GitHub e LinkedIn.

Sobre mim: Texto detalhando minha formação, cursos, interesses e trajetória como estudante de tecnologia.

Currículo: Informações pessoais, contatos, estudos realizados e certificações.

O foco do projeto foi:

Estruturar páginas utilizando HTML semântico;

Criar uma interface limpa, organizada e visualmente agradável com CSS;

Melhorar minha familiaridade com classes, organização de estilos e responsividade básica;

Desenvolver um portfólio funcional para crescer profissionalmente e iniciar minha presença online como desenvolvedor.

Este projeto marca o início da minha jornada na programação e continuará sendo aprimorado com novos recursos, design mais moderno e futuras tecnologias como JavaScript, responsividade avançada, animações e integração com projetos reais.
